A program created through the implementation of a Convolutional Siamese Neural Network model, functioning to 
detect and verify handwritten signatures from large public datasets such as CEDAR and UTSig.


A REFERENCE ON THE FILES WITHIN THIS ZIP FOLDER:

Fraudify_CNN.ipynb : The Phase 1 CNN-based Signature Verification program

Fraudify_SCNN.ipynb : The Phase 2 SCNN-based Signature Verification program
